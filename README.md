# Spelling Bee Practice

A browser-based spelling bee practice app built for kids preparing for the spelling bee at CUSDK8 (Cupertino Union School District). Practice words are sourced from the Santa Clara County Spelling Bee webpage.

## Features

- 20-word practice sessions with text-to-speech pronunciation (American English)
- Selectable difficulty level (Easy through Expert)
- Audio-only hints: request definitions, word origins, and usage in a sentence
- Scoring with streak bonuses
- End-of-session review of all words

## Usage

Open `index.html` in a browser or visit the [live site](https://bhajancodes.github.io/SpellingBee/).
